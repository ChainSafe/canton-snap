# Canton Snap

MetaMask Snap for non-custodial Canton Network signing.

Provides secp256k1 ECDSA signing with SHA-256 hashing and ASN.1 DER encoding — the signature format Canton's Interactive Submission API requires but MetaMask cannot produce natively.

## Status

**Phase 1: Crypto compatibility** — TypeScript crypto modules with cross-validation tests against Go's `pkg/keys/canton_keys.go` from [canton-middleware](https://github.com/ChainSafe/canton-middleware).

## Modules

- **`src/spki.ts`** — Compressed pubkey → X.509 SubjectPublicKeyInfo DER encoding
- **`src/fingerprint.ts`** — SPKI DER → Canton multihash SHA-256 fingerprint
- **`src/sign.ts`** — Pre-hashed digest → ASN.1 DER-encoded ECDSA signature

## Test Vectors

`test/vectors.json` is generated by `canton-middleware`'s `cmd/generate-test-vectors`:

```bash
# In the canton-middleware repo:
make test-vectors
cp test-vectors.json ../canton-snap/test/vectors.json
```

## Development

```bash
npm install
npm test          # Run cross-validation tests against Go vectors
npm run build     # Compile TypeScript
```
